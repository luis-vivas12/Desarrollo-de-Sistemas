1. Iniciamos el repositorio con dos archivos
git init

2. Añadimos archivos
git add .

3. Configuramos el usuario y el correo
git config -- global user.email lvivas*****@utm.edu.ec
git config --global user.username "luis-vivas12"

4. Añadimos direccion del repositorio
git remote add origin https://github.com/luis-vivas12/Desarrollo-de-sistemas.git
5. subimos los archivos
git branch -M main
git push -u origin main
